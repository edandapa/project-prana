# project-prana: The Agent Foundry
Welcome to Project-Prana, a central hub for my autonomous AI agents. This project explores the intersection of Retrieval-Augmented Generation (RAG) and the Model Context Protocol (MCP).

🤖 Included Agents
Flight-Assistant: A real-time travel agent that uses ChromaDB to store personal preferences and the Amadeus API for live flight booking.

🛠️ Tech Stack
Protocol: Model Context Protocol (MCP)
Vector DB: ChromaDB
Language: Python
Interface: Claude Desktop / MCP Inspector
